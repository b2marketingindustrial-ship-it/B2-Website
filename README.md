# b2-marketing-website

Projeto Next.js com Tailwind CSS para site institucional da **B2 Marketing Industrial**.

## Estrutura atual
- Navbar com links `home`, `about`, `process` (use esses nomes no código).
- Logo "b2" em azul néon piscante (`animate-blink`).
- Navbar fixa com `backdrop-blur` e transparência.
- Hero full-screen com título e subtítulo.
- Tema escuro com fundo preto.

## Configuração e execução
1. Instale dependências:
   ```bash
   npm install
   ```
2. Rode em modo desenvolvedor:
   ```bash
   npm run dev
   ```
3. Abra `http://localhost:3000` no navegador.
